# Covid19-Fake-News-Classfier
Using NLP to detect fake news on Covid19. Achieved 89% accuracy on validation dataset.

**Folder Structure**
1. app.py: This file contains the Flask specific code.
2. lstm_model.py: This file contains the original code used to train the model.
3. Templates folder: This folder contains the HTML files. These HTML files will be rendered on the web browser.
4. Static folder: This folder contains CSS files.

**To Run The Application on localhost :**
- Change current directory to the directory containing the files.
- Run 'python app.py' on cmd. This will now run your python application using Flask micro framework on your local machine. 
- Enter the local host http address into browser.
